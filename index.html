MY WORK

  • Poller A site to create/vote/view polls!


    Poller is a polling website where users can create polls to be voted on by other users or guests!

    The homepage's poll tiles constantly change based on trending/popular polls and appear larger when they are even more popular than the rest of the trending tiles! 'Sponsered' polls can be set to be always in a certain positon/size/link out.

    Future improvements include implimenting d3.js to show more statistics/votes over time on results page and poll tiles! Back-end already supports it, just need styled!

    Front-end includes Bootstrap 3, Sass, Masonry, and rendered Jade Pug.

    On the back, Node.js/Express.js/Express-session using a NoSQL MongoDB/Passwordless/EmailJS.

    Lesson Learned?

    Callbacks are extremely powerful and are an amazing tool to create an efficient async back-end (as well as being completely nessesary to accurately work with data passing it along to different controllers). ALSO HAVE A PLAN! A plan for databases, collections, handlers, server/clientside processing, EVERYTHING to be efficient and clean!
